# Commande_Bash/PowerShell :  

```
Get-Alias -Name cp, rm, cd, man, history, cat
```

`cp --> Copy-Item`

`rm --> Remove-Item`

`cd --> Set-Location`

`mkdir -->  New-Item -ItemType Directory`

`man --> help`

`history --> Get-History`

`alias --> Get-Alias`

`cat --> Get-Content`
